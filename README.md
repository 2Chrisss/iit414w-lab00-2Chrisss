Setup instructions:

    1. Create an virtual environment with this command in the terminal:
            python -m venv env

    2. In Windows, write this:
            .venv\Scripts\activate
       In Linux:
            source .venv/bin/activate

    3. To install all the libraries
            pip install -r requirements.txt

    Just Run all the cells on both notebooks. (Run All in VSCode or execute one by one)
